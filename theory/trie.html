Бор (Trie)

Бор - строковый алгоритм, который позволяет поддерживать множество строк и делать с ним различные операции: добавление, удаление, подсчет количества каждой строки, выдавать лексикографически все строки за их суммарную длину.

Здесь бор написан через ссылки, что совершенно не обязательно. Можно хранить все вершины в массиве структур. 
struct trie {

    struct Node{
        int go[26] = {0,};
        int cnt = 0;
        bool flag = 0;
    };

    Node nodes[100500];
    int tsz = 1;

    void Add(string &s) {
        int v = 0;
        for(int i=0;i<s.size();i++) {
            if (nodes[v].go[s[i] - 'a']==0) {
                nodes[v].go[s[i] - 'a'] = tsz++;
            }
            v = nodes[v].go[s[i] - 'a'];
            nodes[v].cnt++;
        }
        nodes[v].flag = 1;
    }

    void Kth(int v,int k) {
        if (k<=0) return;
        if (nodes[v].flag == 1) k--;
        if (k<=0) return;

        for(int i=0;i<26;i++) {
            if (nodes[nodes[v].go[i]].cnt>=k) {
                out.push_back(i+'a');
                Kth(nodes[v].go[i],k);
                break;
            } else k-=nodes[nodes[v].go[i]].cnt;
        }

    }

};

Битовый бор

Это проецирование случая со строками на обычные числа, если представить их в битовой записи. То есть каждое (например интовое) число превращается в строку из 32 нулей и единиц, по которым мы строим бор.

Это позволяет добавить к множеству интовых чисел интересные функции связанные с битовыми операциями. К примеру Найти число в множестве, у которого xor c x максимален. Это лишь одно из многих применений, возможны задачи с and или or, но все-таки xor встречается чаще.
