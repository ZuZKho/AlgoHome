Центроидная декомпозиция

Данный алгоритм используется на деревьях, в которых в течение задачи не меняется структура.

Для меня главное применение этой структуры: найти количество пар вершин, на пути между которых выполняется функция. Например: найти количество пар вершин, таких, что расстояние между ними равно k.

Однако на сайтах с теорией есть и другие применения. Например: посчитать значение функции на пути.

Изучал данную структуру на алгоритмике, мой код полностью совпадает с их кодом. Так же есть пара полезных постов на CF'e.
