Разреженная таблица (Sparse Table)

Разреженная таблица - одна из структур, которая позволяет считать функции на отрезке. По своей сути - это прокаченные префиксные суммы. В отличии от префиксных сумм, она позволяет считать также функции типа min,max и т.д. Но в отличии от ДО не позволяет делать запросы изменений.

Стоит добавить, что она очень проста в написании и понимании, является частью других алгоритмов, например LCA.

Алгоритмика
Cp-algo, на английском. В этой статье снизу очень много задач.

Мой код: 

int sp[100001][20];

int minn(int l,int r) {
    int i=log2(r-l+1);
    return min(sp[l][i],sp[r-(1<<i)+1][i]);
}

void Init(vector<int> &a) {
    int n = a.size();
    for(int i=0;i<n;i++) {
        sp[i][0]=a[i];
    }
    for(int i=1;i<20;i++) {
        for(int l=0;l + (1 << (i-1))<n;l++) {
            sp[l][i]=min(sp[l][i-1],sp[l + (1 << (i-1)) ][i-1]);
        }
    }
}
